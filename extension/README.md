Exact Draughts package

Install by unzipping the release archive at the Kindle USB-storage root.
Launch from KUAL: Exact Draughts -> Exact Draughts.

Artwork credit: Draughts by Thiago Fernandes, https://github.com/tobagin/Draughts
License: GNU GPL version 3 or later.

Modes: Play Red, Play Black, 2 Players, and AI Demo.
Variants: English and International.
AI levels: Easy, Medium, and Hard.
Save and Load use /mnt/us/exact-draughts.save.
Use the right-side move list and |<, <, >, >| buttons to review history.
